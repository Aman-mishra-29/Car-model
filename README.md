This repository contains a complete end-to-end Machine Learning pipeline for Car Model Prediction / Classification. The project demonstrates real-world ML engineering practices including: <br>
Data preprocessing <br>
Feature engineering <br>
Model training & evaluation <br>
Experiment tracking <br>
Reproducible pipeline structure <br>
Production-ready code organization <br>
This project reflects practical ML workflow implementation suitable for Data Science / ML Engineer roles. <br><br>
Problem Statement <br>
The objective of this project is to build a supervised learning model capable of predicting car models based on structured input features such as: <br>
Engine specifications <br>
Fuel type <br>
Transmission <br>
Mileage <br>
Price <br>
Manufacturing attributes <br>
This is treated as a classification problem. <br>
