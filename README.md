**Project Overview** <br>
This repository contains an end-to-end Machine Learning Regression Pipeline built using Linear Regression to predict car prices based on structured input features. <br>
The project demonstrates: <br>
► Data preprocessing <br>
◉ Feature engineering <br>
◉ Model training & evaluation <br>
◉ Experiment tracking <br>
◉ Reproducible pipeline structure <br>
◉ Production-ready code organization <br><br>
**Problem Statement** <br>
The objective of this project is to build a Linear Regression model that predicts the price of a car based on various features such as: <br>
◉ Engine specifications <br>
◉ Fuel type <br>
◉ Transmission <br>
◉ Mileage <br>
◉ Price <br>
◉ Manufacturing attributes <br>
This is treated as a Supervised Learning – Regression Problem. <br><br>
**Dataset Description** <br>
◉ Structured tabular dataset <br>
◉ Contains categorical and numerical features <br>
◉ Target variable: Car Model <br>
◉ Preprocessing includes: <br>

  ◉ Label encoding / One-hot encoding <br>
